## 1. コンセプト

日本のアニメキャラクターがガイド役となり、Vimコマンドをステップバイステップで楽しく学べるインタラクティブな学習サイト。

### ターゲットユーザー
- Vimを初めて学ぶプログラミング初心者
- 楽しく学びたいエンジニア
- アニメが好きな技術者

### 特徴
- キャラクターによる親しみやすいガイド
- 実際にキーを押して学べるインタラクティブ性
- 習熟度に基づく復習システム
- SNSでの成果シェア機能

### スペースド・リピティションアルゴリズム

```javascript
const calculateNextReview = (command) => {
  const { successCount, lastPracticed, consecutiveCorrect } = command;

  // 基本間隔（日数）
  const baseIntervals = [1, 2, 4, 7, 14, 30];

  // 連続正解数に基づく間隔
  const intervalIndex = Math.min(consecutiveCorrect, baseIntervals.length - 1);
  const interval = baseIntervals[intervalIndex];

  // 次回復習日を計算
  const nextReview = addDays(lastPracticed, interval);

  return nextReview;
};

const getCommandsNeedingReview = (commands) => {
  const today = new Date();
  return commands
    .filter(cmd => cmd.nextReview <= today)
    .sort((a, b) => a.masteryLevel - b.masteryLevel);
};
```

### キーボード機能一覧

| 機能 | 説明 | 実装方法 |
|------|------|---------|
| 学習キーハイライト | 今のレッスンで使うキーを色付け | CSS class切り替え |
| リアルタイム点灯 | 押したキーが光る（0.2秒） | keydown イベント + CSS animation |
| 入力履歴表示 | 直近10回の入力を表示 | 配列で管理 |
| モディファイア対応 | Ctrl, Shift などの組み合わせも表示 | 複合キー検知 |
| キー説明ツールチップ | ホバーでVimコマンド説明表示 | title属性 or カスタムツールチップ |
| 正解/不正解表示 | 正解は緑、不正解は赤で点灯 | 条件分岐でクラス付与 |

### キーボードコンポーネント設計

```javascript
const Keyboard = ({ activeKeys, pressedKey, keyHistory }) => {
  const rows = [
    ['`', '1', '2', '3', '4', '5', '6', '7', '8', '9', '0', '-', '='],
    ['q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p', '[', ']', '\\'],
    ['a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l', ';', "'"],
    ['z', 'x', 'c', 'v', 'b', 'n', 'm', ',', '.', '/']
  ];

  return (
    <div className="keyboard">
      {rows.map((row, i) => (
        <div key={i} className="keyboard-row">
          {row.map(key => (
            <Key
              key={key}
              keyChar={key}
              isActive={activeKeys.includes(key)}
              isPressed={pressedKey === key}
              command={getCommandForKey(key)}
            />
          ))}
        </div>
      ))}
      <KeyHistory history={keyHistory} />
    </div>
  );
};
```

## 12. ページ構成

### ルーティング

```
/vim-tutorial
├── /                      → トップページ（レッスン一覧）
├── /lesson/:id            → 各レッスンページ
│   ├── /lesson/1          → 基本移動
│   ├── /lesson/2          → 単語移動
│   └── ...
├── /practice              → フリー練習モード
├── /commands              → 覚えたコマンド一覧
├── /review                → 復習モード
├── /progress              → 進捗確認ページ
├── /achievements          → 実績一覧
└── /share/:achievementId  → シェア用ページ（OGP対応）
```

| コマンド一覧 | 習得コマンドの管理 | CommandTable, FilterBar |
| 復習 | スペースドリピティション | ReviewCard, ProgressBar |
| 進捗 | 詳細な統計情報 | Charts, AchievementList |
| シェア | SNS向けページ | ShareCard, OGPMeta |

## 付録

### コマンド一覧（実装予定）

#### 移動系
| コマンド | 説明 |
|---------|------|
| h | 左に移動 |
| j | 下に移動 |
| k | 上に移動 |
| l | 右に移動 |
| w | 次の単語の先頭へ |
| b | 前の単語の先頭へ |
| e | 単語の末尾へ |
| 0 | 行頭へ |
| $ | 行末へ |
| ^ | 行の最初の非空白文字へ |
| gg | ファイル先頭へ |
| G | ファイル末尾へ |
| {数字}G | 指定行へ |

#### 編集系
| コマンド | 説明 |
|---------|------|
| i | カーソル位置で挿入モード |
| a | カーソルの後ろで挿入モード |
| o | 下に新しい行を作って挿入モード |
| O | 上に新しい行を作って挿入モード |
| x | 1文字削除 |
| dd | 行削除 |
| dw | 単語削除 |
| yy | 行コピー |
| yw | 単語コピー |
| p | 下/後ろにペースト |
| P | 上/前にペースト |
| u | アンドゥ |
| Ctrl+r | リドゥ |

#### 検索・置換系
| コマンド | 説明 |
|---------|------|
| /{pattern} | 前方検索 |
| ?{pattern} | 後方検索 |
| n | 次の検索結果 |
| N | 前の検索結果 |
| :s/old/new/ | 置換（現在行） |
| :%s/old/new/g | 全置換 |

#### その他
| コマンド | 説明 |
|---------|------|
| v | ビジュアルモード（文字単位） |
| V | ビジュアルモード（行単位） |
| Ctrl+v | ビジュアルモード（矩形） |
| . | 直前の操作を繰り返し |
| q{register} | マクロ記録開始 |
| @{register} | マクロ実行 |
| :w | 保存 |
| :q | 終了 |
| :wq | 保存して終了 |
| :q! | 保存せず強制終了 |
